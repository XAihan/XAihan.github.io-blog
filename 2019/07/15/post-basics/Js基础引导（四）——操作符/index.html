你好，朋友。
欢迎进入JS基础引导——操作符

概述

操作符，说简单点，就是协助我们进行运算（操作数据值）的操作符。
包括：算数操作符，位操作符，关系操作符，相等操作符
PS：当应用于对象是，相应的操作符通常会调用对象的valueof（）方法和（或）toString（）方法。

一元操作符

只能操作一个值的操作符就是一元操作符。
顾名思义，一元操作符的使用对象只有一个。

常用符号：
++(数值加1) - -（数值减一） +（原来值不变） -（原来值的负数）
